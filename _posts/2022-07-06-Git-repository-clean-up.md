## repository 병합하기
---
git을 본격적으로 사용하다보니 처음 git을 시작했을때 생각없이 올린 repository를 정리해 카테고리별로 합치고 싶어졌다. repository들을 모두 clone하여 하나의 폴더에 합치고 다시 push하자니 너무 일이 많아질 것 같아 repository들을 병합하는 법을 공부해 해보았다.
